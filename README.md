### Img-to-webp Converter

一个可以将 jpg|jpeg|png|gif 转为 webp 格式的小工具。

### 使用方法：

#### 克隆到本地

```bash
git clone https://github.com/cheungray123/img-to-webp.git
```

#### 安装依赖

```bash
pnpm install
```

#### 运行

```bash
pnpm webp
```
